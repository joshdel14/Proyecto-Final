Dos routers unidos


El objetivo de esta práctica es crear grupos de trabajos para la transmisión de datos mediante un router. Pasos para realizar esta práctica: 1.- Primero creamos dos Switch asignando cinco ordenadores a cada uno.
2.- Utilizamos un router para la interconexión entre Switch.
3.- Asignamos las IP a cada una de las maquinas, recordando que en esta ocasión al utilizar un router de por medio así que debemos especificar con el Switch #1 la IP 192.168.1.1 y para el Switch #2 la IP 192.168.2.1.
4.- Debemos asignar el Gateway de la siguiente manera.
5.- Conectamos los equipos con su respectivo Switch mediante cable directo.

6.- Luego procedemos a la conexión con el router mediante cable directo.
7.- Para poder concretar la conexión exitosa, no olvidemos especificar la dirección MAC, la dirección IP, la máscara de la Subred y asegurarnos que el Estado del Puerto este ENCENDIDO.
8.- Para comprobar la conexión realizamos un envió de paquetes y listo.
