Colores


Los colores web son aquellos colores que aparecen en una página web. Se pueden basar sobre los sistemas de color RGB o HSL. En el código CSS (y antiguamente en HTML) son especificados como valores numéricos, aunque hay colores que también pueden utilizarse por sus nombres en inglés.

Bordes


Existen dos formas para crear una línea separadora horizontal y una única forma para crear una línea separadora vertical. Para crear una línea horizontal la forma fácil y rápida es utilizar la etiqueta hr, que no se cierra y que crea una línea horizontal simple. Esta etiqueta admite algunas características CSS como el grosor de la línea (border-width), la anchura (width), el color (border-color) o el tipo de línea (border-style). Para representar la línea azul punteada que hay sobre este párrafo es necesario tanto la etiqueta html como las declaraciones CSS: hr En este ejemplo el selector CSS (hr{) hace referencia a todas las líneas de la página. En el caso de querer hacer referencia únicamente a una, la etiqueta tendría que tener un id para poder identificarla y el selector CSS sería.

Con la propiedad border-radius tenemos la posibilidad de crear div con las esquinas con bordes redondeados, pudiendo especificar la suavidad de las esquinas de manera conjunta o cada una de las esquinas por separado. Para poder visualizar una esquina con borde redondeado, además de la propiedad border-radius, tenemos que definir un borde normal (con border) y/o indicar un color de fondo (con background-color).



Gradient


CSS Gradient es un pequeño sitio web feliz y una herramienta gratuita que le permite crear un fondo degradado para sitios web. Además de ser un generador de gradientes css, el sitio también está repleto de contenido colorido sobre gradientes, desde artículos técnicos hasta ejemplos de gradientes de la vida real como Stripe e Instagram.

Mira, los gradientes se jugaron mucho en los primeros días de la web, pero ahora son tan ubicuos que sería negligente no dejarlos caer en tu sitio, interfaz o próximo trabajo de tinte de cabello. Además, formo parte de un grupo de creadores con la misión de construir una Internet mejor, un proyecto digital a la vez. Uno de nuestros lanzamientos de proyectos recientes es Cool Backgrounds , otra herramienta de diseño gratuita para generar fondos de pantalla para sitios web, blogs y teléfonos.


Caja de sombras


Todo lo que sabemos acerca de text-shadowstar puede ser aplicado a box-shadow. Es como si fuera la misma propiedad con otro nombre. Sin embargo hay algunos toques especiales. Empecemos con la sintaxis.

Pero no siempre tenemos cajas regulares. Pensemos un momento en un bocadillo ( como elemento de viñeta ). En el siguiente ejemplo para dibujar un bocadillo utilizamos una caja rectangular y un pseudo-elemento ::before para dibujar la flecha. Si queremos ponerle una sombra normal ( box- shadow:1px 1px 2px #333; ) la flecha queda sin sombra. Podemos darle una sombra al ::before, pero en este caso la sombra aparece marcada por encima del bocadillo

Mientras que box-shadow aplica la sombra al contorno del elemento, filter:drop-shadow() toma en cuenta las transparencias.
